====================
Steps to install
====================

git clone https://github.com/ranidu-harshana/photographer_information_system.git
cd photographer_information_system
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate

create a databe ---> photographer_information_system
run sql queries in sql_queries.txt

php artisan serve
